# Muhammad Talha - Flutter Portfolio

Personal portfolio built with Flutter Web to showcase experience, projects, and certifications.

## Live Demo

Set your GitHub Pages URL once deployed, for example:
`https://<username>.github.io/my-portfolio/#/`

## Highlights

- Modern, responsive Flutter UI with a custom visual system
- Project showcase with detailed summaries and links
- Resume section covering experience, education, and core skills
- Certifications grid with credential links when available

## Tech Stack

- Flutter Web
- Firebase
- Node.js / Express
- GetX, Google Fonts, Flutter SVG, URL Launcher

## Run Locally

```bash
flutter pub get
flutter run -d chrome
```

## Deploy to GitHub Pages

Build with your repository name as the base href:

```bash
flutter build web --base-href "/my-portfolio/"
```

Then deploy the `build/web` output to GitHub Pages.
